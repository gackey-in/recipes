# 豆苗の煮浸し
- ４人分

## 材料
- 豆苗：２束
- もやし：１袋
- にんにく：1/2片
- 鷹の爪：１本
- ごま油：大さじ１
- 塩：適量
- 出汁：480cc
- みりん：30cc
- 醤油：30cc

## つくりかた
1. 豆苗は根を取り、２等分にする。もやしはひげ根を取る。
2. にんにくを潰し、唐辛子は種を取る。
3. 鍋に出汁・みりん・醤油を合わせ、浸し汁をつくっておく。
4. 別の鍋に胡麻油とにんにくを入れて火にかけ、香りが出たら唐辛子を入れる。
5. 野菜を入れて炒め、浸し汁を加えてひと煮立ちさせる。そのまま火を止め、味を見て足りなければ塩を足す。

## :memo: 
- 煮浸しの浸し汁は出汁：しょうゆ：みりん = 16：1：1
